## Documentation

Node JS, Gemini AI Based PR Review Application

### How to setup application ?

```sh
1. Create an API Key using url. (https://aistudio.google.com/app/apikey)
2. Create .env file in the application and add Gemini AI API Key like this:
GEMINI_AI_API_KEY=your api key
3. Generate GitHub Token from your own account.
4. Execute command in application root 'npm install'
5. Install nodemon package in your machine using command 'npm install -g nodemon'
6. Execute command 'nodemon start'
```
